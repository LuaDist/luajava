Histórico

1.1 [24/jan/2007]
  • mudanças para se adaptar a Lua 5.1;
  • incluiu LUA_MULTRET como constante de LuaState;
  • corrigido bug em LuaObject.call na ordem de retorno dos resultados da função.
1.0 [22/jun/2005]
  • aperfeiçoamento do tratamento de erros;
  • abandonada a compatibilidade com JDK 1.3;
  • binários do Windows compatíveis com o LuaBinaries 5.0.2 Release 2.
  • novo formato da documentação.
1.0 Beta 4 [21/jan/2005]
  • pacote renomeado como "org.keplerproject.luajava";
  • LuaObject.getLuaState passou a ser público;
  • nova função LuaObject.type();
  • JavaFunction.foo() renomeado como JavaFunction.execute();
  • método chamado por javaLoadLib passa a poder retornar um int.
1.0 Beta 3 [21/set/2004]
  • nova função em lua luajava.loadLib, que abre bibliotecas para Lua escritas em Java;
  • função LuaState.setGlobal passa a aceitar String, para funcionar como Lua setglobal;
  • JavaFunction.foo() gera uma LuaException;
  • nova função LuaState.pushString(byte[]);
  • Problemas na vinculação de algumas funções de lua ao Java;
  • correções de pequenos problemas.
1.0 Beta 2 [07/jul/2004]
  • passa a oferecer suporte a vários threads;
  • nova função LuaState.close() que deve ser chamada para fechar o estado;
  • LuaState não tinha algumas funções;
  • correções de vários problemas.
1.0 Beta 1 [28/jun/2004]
